## What is Tuim?

Tuim is project that aims to provide their own ABI
and their own standard library,
focusing code portability and language interoperability.

## Why use Tuim?

- Portability:

  Code matching Tuim's ABI may be executed on different operating systems
  without the need of a virtual machine
  and code writen using Tuim's standard library may be compiled for
  different architectures and operating systems.
